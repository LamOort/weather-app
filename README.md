# Weather App

A weather dashboard built with React, TypeScript, Vite, TanStack Query, and SCSS. Features real-time weather data via Open-Meteo, location search, and unit toggling.

## 🚀 Quick Start

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) (v22 or higher) installed.

### 2. Installation
Clone the repository and install dependencies:
```bash
npm install
```

### 3. Running Locally
Start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

## 🛠️ Available Scripts

- `npm run dev` - Start development server with HMR.
- `npm run build` - Build the application for production (to the `dist/` folder).
- `npm run preview` - Locally preview the production build.
- `npm run test` - Run the test suite using Vitest.
- `npm run lint` - Run ESLint to check for code quality issues.

## 🧰 Tech Stack
- **Frontend**: React 19, TypeScript, Vite
- **State Management**: TanStack Query (React Query)
- **Styling**: SCSS (CSS Modules/Variables)
- **Icons**: React Icons (Fa, Lu)
- **API**: Open-Meteo API
